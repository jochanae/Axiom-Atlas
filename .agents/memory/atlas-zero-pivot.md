On June 16, 2026, after a full diagnostic session, the project reached a strategic reset.

**Root cause of all failures:** Not broken code. Over-connection. Too many systems (Lovable + Cloud Run + Supabase + two GitHub repos + Replit) with no single source of truth. Every fix in one system broke another.

**ATLAS_ZERO.md** lives at the workspace root (`/home/runner/workspace/ATLAS_ZERO.md`). It is the constitution. It wins over any code, feature, or conversation that conflicts with it.

**The governance rule:** If code, features, architecture, or conversations conflict with ATLAS_ZERO.md, ATLAS_ZERO.md wins.

**Phase 1 — the only three things that matter:**
1. Turn an idea into a project (Understand it)
2. Turn the project into something working (Build it)
3. Continue building from where they left off (Evolve it)

**The Atlas Rule:** If a new idea does not directly improve one of those three actions, it waits.

**The Founder Filter:** Before any feature: Does this help understand an idea? Build something working? Continue later without starting over? No to all three = it waits.

**What this means for future build sessions:**
- Before writing any code, check against ATLAS_ZERO.md
- The core loop (Idea → Build → Continue) must work before anything else gets added
- One environment. One source of truth. No split architecture.
- Atlas earns complexity. Complexity is no longer free.

**Next step when Jochanae returns:** She will bring "Atlas Zero → Atlas One" — a minute-by-minute 10-minute user experience document. That becomes the Phase 1 build spec.

**What was NOT said:** The existing visual identity (Luxury Obsidian) survives. The personality survives. The promise survives. The code must re-earn its place.
