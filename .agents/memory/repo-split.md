name: Repo split
description: Which GitHub repo is backend vs frontend for Axiom-Atlas
---

**Backend repo:** `jochanae/Axiom-Atlas` — this Replit. Express API server, deployed to Google Cloud Run.

**Frontend repo:** `jochanae/atlas-idk` — separate repo, managed via Lovable and Cursor Agent on mobile.

**Why:** Jochanae confirmed this directly. replit.md previously only mentioned jochanae/Axiom-Atlas and called it the full repo, but the frontend lives separately at atlas-idk.

**How to apply:** Any Cursor/Lovable prompts for frontend changes target atlas-idk. Any backend changes are made here in this Replit and pushed to Axiom-Atlas via the Git tab.
